## 全体のユースケース

### 認証・アカウント管理

- ユーザーとして、サインアップ画面からメールアドレスとパスワードでアカウントを作成したい
- ユーザーとして、ログイン画面でメールアドレスとパスワードを入力してログインしたい
- ユーザーとして、ログアウトして他のアカウントに切り替えたい
- ユーザーとして、未ログイン状態ではレビュー一覧にアクセスできないようにしたい

### レビュー投稿・管理

- ユーザーとして、読んだ本の書籍名・感想・星評価・読書ステータスを記録したい
- ユーザーとして、レビューに写真（最大4枚）を添付して視覚的に記録を残したい
- ユーザーとして、全ユーザーのレビューを新着順で見たい（全体フィード）
- ユーザーとして、フォロー中のユーザーのレビューだけを見たい（フォローフィード）
- ユーザーとして、自分のレビューを編集したい
- ユーザーとして、自分のレビューを削除したい
- ユーザーとして、レビューの詳細（コメント・いいね数）を確認したい

### いいね・コメント

- ユーザーとして、参考になったレビューにいいねしたい
- ユーザーとして、いいねを取り消したい
- ユーザーとして、各レビューに何件いいねがついているか確認したい
- ユーザーとして、レビューにコメントして感想を伝えたい
- ユーザーとして、自分のコメントを削除したい
- ユーザーとして、各レビューに何件コメントがついているか確認したい

### プロフィール・フォロー

- ユーザーとして、自分のプロフィール（ユーザー名・アイコン・自己紹介）を編集したい
- ユーザーとして、他のユーザーのプロフィールと投稿レビュー一覧を確認したい
- ユーザーとして、気になるユーザーをフォローして読書仲間を増やしたい
- ユーザーとして、フォローを解除したい
- ユーザーとして、自分のフォロー一覧・フォロワー一覧を確認したい
- ユーザーとして、レビューのユーザー名をクリックしてプロフィールを確認しフォローしたい
