# 画面設計書

## 1. 画面一覧

| # | 画面名 | URL | 認証必須 |
|---|--------|-----|----------|
| 1 | サインアップ画面 | `/signup` | 不要 |
| 2 | ログイン画面 | `/login` | 不要 |
| 3 | レビュー一覧画面 | `/` | 必要 |
| 4 | レビュー詳細画面 | `/reviews/:id` | 必要 |
| 5 | レビュー作成画面 | `/reviews/new` | 必要 |
| 6 | レビュー編集画面 | `/reviews/:id/edit` | 必要（投稿者のみ） |
| 7 | プロフィール画面 | `/users/:id` | 必要 |
| 8 | プロフィール編集画面 | `/users/:id/edit` | 必要（本人のみ） |
| 9 | フォロー一覧画面 | `/users/:id/following` | 必要 |
| 10 | フォロワー一覧画面 | `/users/:id/followers` | 必要 |

## 2. 画面遷移図

```
[サインアップ画面 /signup]
  ├── 登録成功 → [レビュー一覧画面 /]
  └── ログインリンク → [ログイン画面 /login]

[ログイン画面 /login]
  ├── ログイン成功 → [レビュー一覧画面 /]
  └── 新規登録リンク → [サインアップ画面 /signup]

[レビュー一覧画面 /]（ホーム）
  ├── [全体] / [フォロー中] タブ切り替え（ページ内）
  ├── レビュー作成ボタン → [レビュー作成画面 /reviews/new]
  ├── レビューカードクリック → [レビュー詳細画面 /reviews/:id]
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  └── ログアウト → [ログイン画面 /login]

[レビュー作成画面 /reviews/new]
  ├── 投稿成功 → [レビュー一覧画面 /]
  └── キャンセル → [レビュー一覧画面 /]

[レビュー編集画面 /reviews/:id/edit]
  ├── 保存成功 → [レビュー詳細画面 /reviews/:id]
  └── キャンセル → [レビュー詳細画面 /reviews/:id]

[レビュー詳細画面 /reviews/:id]
  ├── コメント投稿（その場で更新）
  ├── いいねボタン（その場で更新）
  ├── 編集ボタン（本人のみ） → [レビュー編集画面 /reviews/:id/edit]
  ├── 投稿者ユーザー名クリック → [プロフィール画面 /users/:id]
  ├── コメント投稿者ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [レビュー一覧画面 /]

[プロフィール画面 /users/:id]
  ├── フォローボタン（その場で更新）
  ├── フォロー数クリック → [フォロー一覧画面 /users/:id/following]
  ├── フォロワー数クリック → [フォロワー一覧画面 /users/:id/followers]
  ├── 編集ボタン（本人のみ） → [プロフィール編集画面 /users/:id/edit]
  ├── レビューカードクリック → [レビュー詳細画面 /reviews/:id]
  └── 戻るボタン → [レビュー一覧画面 /]

[プロフィール編集画面 /users/:id/edit]
  ├── 保存成功 → [プロフィール画面 /users/:id]
  └── キャンセル → [プロフィール画面 /users/:id]

[フォロー一覧画面 /users/:id/following]
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [プロフィール画面 /users/:id]

[フォロワー一覧画面 /users/:id/followers]
  ├── ユーザー名クリック → [プロフィール画面 /users/:id]
  └── 戻るボタン → [プロフィール画面 /users/:id]
```

## 3. 各画面の詳細

### 3.1 サインアップ画面（`/signup`）

**目的**: 新規ユーザーのアカウント作成

**ワイヤーフレーム**

```
+--------------------------------------------+
|              BookLog                       |
+--------------------------------------------+
|  ユーザー名  [________________]            |
|  メールアドレス [________________]         |
|  パスワード  [________________]            |
|  パスワード確認 [________________]         |
|                                            |
|          [  アカウント登録  ]              |
|                                            |
|  すでにアカウントをお持ちの方は ログイン   |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | バリデーション |
|------|------|--------------|
| ユーザー名 | テキスト入力 | 必須・50文字以内 |
| メールアドレス | メール入力 | 必須・メール形式 |
| パスワード | パスワード入力 | 必須・8文字以上 |
| パスワード確認 | パスワード入力 | 必須・パスワードと一致 |
| アカウント登録ボタン | ボタン | バリデーション後に送信 |
| ログインへのリンク | テキストリンク | 既存ユーザー向け |

---

### 3.2 ログイン画面（`/login`）

**目的**: 既存ユーザーの認証

**ワイヤーフレーム**

```
+--------------------------------------------+
|              BookLog                       |
+--------------------------------------------+
|  メールアドレス [________________]         |
|  パスワード  [________________]            |
|                                            |
|            [  ログイン  ]                  |
|                                            |
|  アカウントをお持ちでない方は 新規登録     |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | バリデーション |
|------|------|--------------|
| メールアドレス | メール入力 | 必須 |
| パスワード | パスワード入力 | 必須 |
| ログインボタン | ボタン | |
| 新規登録へのリンク | テキストリンク | |

---

### 3.3 レビュー一覧画面（`/`）

**目的**: 全体フィードまたはフォローフィードのレビューを閲覧する

**ワイヤーフレーム**

```
+--------------------------------------------+
| BookLog          ユーザー名  ログアウト     |
+============================================+
|  [ 全体 ]  [ フォロー中 ]                  |
+--------------------------------------------+
|                    [ レビューを書く ]      |
+--------------------------------------------+
|  ┌──────────────────────────────────────┐  |
|  │ [アイコン] ユーザー名    2026/06/28  │  |
|  │ 📚 書籍名                            │  |
|  │ ★★★★☆  [読了]                      │  |
|  │ 感想の冒頭テキスト...                 │  |
|  │ [写真1][写真2]                        │  |
|  │ ♡ 12        💬 3                     │  |
|  └──────────────────────────────────────┘  |
|  ┌──────────────────────────────────────┐  |
|  │ ...（繰り返し）                       │  |
|  └──────────────────────────────────────┘  |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| ヘッダー | 固定ナビゲーション | ロゴ・ユーザー名・ログアウトボタン |
| フィードタブ | タブコンポーネント | 全体／フォロー中を切り替え（デフォルト：全体） |
| レビューを書くボタン | ボタン | レビュー作成画面へ遷移 |
| レビューカード | カードコンポーネント | クリックでレビュー詳細画面へ |
| ユーザー名（カード内） | リンク | クリックでプロフィール画面へ遷移 |
| 星評価 | 星アイコン表示 | ★で塗りつぶし表示 |
| 読書ステータス | バッジ | 読了 / 読書中 |
| いいねボタン | トグルボタン | カード上でいいね可能 |
| コメント数 | テキスト表示 | クリックでレビュー詳細画面へ |

---

### 3.4 レビュー作成画面（`/reviews/new`）・レビュー編集画面（`/reviews/:id/edit`）

**目的**: レビューを作成・編集する

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る         レビューを書く             |
+--------------------------------------------+
|  書籍名  [________________________________] |
|                                            |
|  星評価  ☆ ☆ ☆ ☆ ☆                       |
|                                            |
|  読書ステータス                            |
|  ( ) 読書中  ( ) 読了                      |
|                                            |
|  感想                                      |
|  +--------------------------------------+  |
|  | この本を読んで...                     |  |
|  |                                      |  |
|  +--------------------------------------+  |
|                                            |
|  [写真1] [写真2] [写真3] [写真4]          |
|  [📷 写真を追加]                           |
|                                            |
|  [キャンセル]           [  投稿する  ]    |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| 書籍名 | テキスト入力 | 必須・200文字以内 |
| 星評価 | クリック可能な星アイコン | 必須・1〜5でクリック選択 |
| 読書ステータス | ラジオボタン | 必須・読書中 / 読了 |
| 感想 | 多行テキスト入力 | 必須 |
| 写真添付ボタン | ファイル選択 | 任意・最大4枚・JPEG/PNG |
| 写真プレビュー | サムネイル表示 | 個別削除可能 |
| 投稿する / 保存ボタン | ボタン | バリデーション後に送信 |
| キャンセルボタン | ボタン | 作成時は一覧・編集時は詳細へ戻る |

---

### 3.5 レビュー詳細画面（`/reviews/:id`）

**目的**: レビューの全文・写真・コメントを表示し、いいね・コメントを行う

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る                                    |
+--------------------------------------------+
| [アイコン] ユーザー名*        2026/06/28  |
| 📚 書籍名                                  |
| ★★★★☆  [読了]                            |
| 感想の全文...                              |
| [写真1][写真2][写真3][写真4]               |
|                                            |
| ♡ 12  [いいねする]     💬 3              |
|                       [編集] [削除]       |
+--------------------------------------------+
| コメント（3件）                            |
|  [アイコン] ユーザー名*  コメント本文 [削除]|
|  [アイコン] ユーザー名*  コメント本文      |
+--------------------------------------------+
| [アイコン] [コメントを入力...]  [送信]     |
+--------------------------------------------+
  * ユーザー名クリック → プロフィール画面へ遷移
```

---

### 3.6 プロフィール画面（`/users/:id`）

**目的**: 特定ユーザーの情報とその投稿レビュー一覧を表示する

**ワイヤーフレーム**

```
+--------------------------------------------+
| ← 戻る                                    |
+--------------------------------------------+
|       [アイコン画像]                       |
|       ユーザー名          [フォロー]       |
|                        （本人：[編集]）    |
|       自己紹介文                           |
|       フォロー: 12  フォロワー: 34         |
+--------------------------------------------+
| 投稿レビュー一覧                           |
|  ┌──────────────────────────────────────┐  |
|  │ レビューカード...                     │  |
|  └──────────────────────────────────────┘  |
+--------------------------------------------+
```

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| アイコン画像 | 画像表示 | S3からロード |
| ユーザー名 | テキスト表示 | |
| 自己紹介文 | テキスト表示 | |
| フォロー数 | テキストリンク | クリックでフォロー一覧へ |
| フォロワー数 | テキストリンク | クリックでフォロワー一覧へ |
| フォローボタン | トグルボタン | 他ユーザーのみ表示 |
| 編集ボタン | ボタン | 本人のみ表示 |
| 投稿レビュー一覧 | カードリスト | そのユーザーのレビューのみ |

---

### 3.7 プロフィール編集画面（`/users/:id/edit`）

**目的**: 自分のプロフィール情報を更新する

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| アイコン画像プレビュー | 画像表示 | 現在のアイコンを表示 |
| アイコン変更ボタン | ファイル選択 | JPEG/PNG・5MB以内・S3にアップロード |
| ユーザー名 | テキスト入力 | 必須・50文字以内 |
| 自己紹介文 | 多行テキスト入力 | 任意・160文字以内 |
| 保存ボタン | ボタン | |
| キャンセルボタン | ボタン | プロフィール画面へ戻る |

---

### 3.8 フォロー一覧画面（`/users/:id/following`）

**目的**: そのユーザーがフォローしているユーザーの一覧を表示する

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| ユーザーカード（繰り返し） | カード | アイコン・ユーザー名・フォローボタン |
| フォローボタン | トグルボタン | 自分自身の行には非表示 |

---

### 3.9 フォロワー一覧画面（`/users/:id/followers`）

**目的**: そのユーザーをフォローしているユーザーの一覧を表示する

**UI要素**

| 要素 | 種別 | 備考 |
|------|------|------|
| ユーザーカード（繰り返し） | カード | アイコン・ユーザー名・フォローボタン |
| フォローボタン | トグルボタン | 自分自身の行には非表示 |
