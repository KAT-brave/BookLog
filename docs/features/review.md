# 機能定義書：読書レビュー投稿

## 概要

ログイン済みユーザーが書籍名・感想・星評価（1〜5）・読書ステータス（読書中／読了）・写真（最大4枚）でレビューを作成・編集・削除できる。写真はAWS S3にpresigned URL方式でアップロードする。

---

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | レビュー作成 | 書籍名・感想・星評価・読書ステータス・写真でレビューを投稿する |
| 2 | 写真添付 | レビュー作成・編集時に写真を最大4枚まで添付できる |
| 3 | 写真プレビュー | 選択した写真をレビュー投稿前にサムネイルでプレビュー表示する |
| 4 | 星評価 | 1〜5の星評価を設定する（必須） |
| 5 | 読書ステータス | reading（読書中）/ completed（読了）を設定する（必須） |
| 6 | レビュー編集 | 自分のレビューの内容・写真を編集できる |
| 7 | レビュー削除 | 自分のレビューのみ削除できる |
| 8 | レビュー詳細表示 | レビュー内容・写真・コメント一覧・いいね数を表示する |

---

## 詳細仕様

### レビュー作成

**入力項目**

| フィールド | 型 | バリデーション |
|-----------|-----|--------------|
| 書籍名 | 文字列 | 必須・200文字以内 |
| 感想 | テキスト | 必須・空白のみ不可 |
| 星評価 | 整数 | 必須・1〜5の整数 |
| 読書ステータス | 文字列 | 必須・reading または completed |
| 写真 | ファイル（複数） | 任意・最大4枚・JPEG/PNG・1枚10MB以内 |

**処理フロー**

1. ユーザーがレビュー作成画面でフォームを入力・写真を選択
2. レビュー投稿ボタンクリック時、フロントエンドでバリデーション
3. 写真がある場合、`POST /api/v1/reviews/presigned_url` でS3 presigned URLを取得
4. 取得したpresigned URLへ写真を直接PUTアップロード（S3へ直接転送）
5. `POST /api/v1/reviews` でレビュー内容とS3写真URLを送信
6. バックエンドがreviewsテーブルとreview_imagesテーブルに保存
7. レビュー一覧画面に戻り、新レビューを先頭に表示

---

### 星評価

- 1〜5の整数で評価（必須）
- フロントエンドではクリック可能な星アイコン（☆）で表示
- 選択した評価値（1〜5）が塗りつぶし星（★）で表示される
- レビュー一覧・詳細画面にも星アイコンで表示

---

### 読書ステータス

| ステータス | 表示名 | 説明 |
|-----------|--------|------|
| reading | 読書中 | 現在読んでいる書籍 |
| completed | 読了 | 読み終わった書籍 |

- レビュー作成時に必須で選択
- レビューカード・詳細画面にバッジ形式で表示
- 後から編集で変更可能

---

### 写真アップロード（S3 presigned URL方式）

```
1. クライアント → POST /api/v1/reviews/presigned_url（ファイル名・枚数を送信）
2. Rails API → AWS SDKでS3 presigned URLを発行
3. クライアント → presigned URLへ直接PUTアップロード（S3へ直接）
4. アップロード完了後、S3のURLをリスト化
5. POST /api/v1/reviews（または PATCH /api/v1/reviews/:id）へ内容 + S3 URLリストを送信
6. バックエンド → review_imagesテーブルにURL・表示順を保存
```

**S3バケット・パス**

| 用途 | バケット | パス例 |
|------|---------|--------|
| レビュー写真 | booklog-review-images | `review-images/reviews/{review_id}/{uuid}.jpg` |

**写真制約**

| 項目 | 制約 |
|------|------|
| 枚数 | 1レビューにつき最大4枚 |
| ファイル形式 | JPEG / PNG |
| ファイルサイズ | 1枚あたり最大10MB |
| 表示順 | 添付した順番（position: 1〜4）で保存・表示 |

---

### レビュー編集

- 自分が作成したレビューのみ編集可能
- 編集画面に現在の内容・写真を初期値として表示
- 写真の個別削除・追加が可能
- 削除した写真はバックエンドでS3上のファイルも削除

---

### レビュー削除

- 自分が作成したレビューのみ削除ボタンを表示
- 削除前に確認ダイアログを表示
- 削除成功後、S3上の添付写真も同時削除（AWS SDK使用）

---

### レビュー詳細表示

- 書籍名
- 感想（全文）
- 星評価（★アイコン）
- 読書ステータス（バッジ）
- 添付写真（最大4枚、S3からロード）
- いいね数・いいねボタン
- コメント一覧
- 投稿者のアイコン・ユーザー名（クリックでプロフィール画面へ遷移）
- 自分のレビューには編集ボタン・削除ボタンを表示

---

## APIエンドポイント

| メソッド | エンドポイント | 認証 | 説明 |
|---------|--------------|------|------|
| GET | `/api/v1/reviews` | 必要 | レビュー一覧取得（全体 or フォロー中） |
| GET | `/api/v1/reviews/:id` | 必要 | レビュー詳細取得 |
| POST | `/api/v1/reviews` | 必要 | レビュー作成 |
| PATCH | `/api/v1/reviews/:id` | 必要（投稿者のみ） | レビュー編集 |
| DELETE | `/api/v1/reviews/:id` | 必要（投稿者のみ） | レビュー削除 |
| POST | `/api/v1/reviews/presigned_url` | 必要 | S3 presigned URL発行 |

---

## エラーケース

| ケース | HTTPステータス | エラーメッセージ例 |
|--------|-------------|----------------|
| 書籍名が空 | 422 | 書籍名を入力してください |
| 書籍名が200文字超過 | 422 | 書籍名は200文字以内で入力してください |
| 感想が空 | 422 | 感想を入力してください |
| 星評価が範囲外 | 422 | 星評価は1〜5で入力してください |
| ステータスが不正 | 422 | 読書ステータスを選択してください |
| 写真5枚以上 | 422 | 写真は最大4枚まで添付できます |
| 非対応ファイル形式 | 422 | JPEG/PNG形式の画像のみ添付できます |
| ファイルサイズ超過 | 422 | 画像のサイズは10MB以下にしてください |
| 他人のレビューを編集 | 403 | この操作は許可されていません |
| 他人のレビューを削除 | 403 | この操作は許可されていません |
| レビューが存在しない | 404 | レビューが見つかりません |
| S3アップロード失敗 | 500 | 画像のアップロードに失敗しました |
