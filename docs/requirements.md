# 要件定義書

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | BookLog |
| 目的 | 学習目的の読書記録Webアプリ |
| 対象ユーザー | 読書好きのユーザー（複数ユーザーの同時利用を想定） |
| フロントエンド | React（TypeScript） |
| バックエンド | Ruby on Rails 8（APIモード） |
| データベース | PostgreSQL |
| 画像ストレージ | AWS S3 |
| 認証方式 | メールアドレス + パスワード（devise-jwt） |

---

## 2. 機能一覧

機能の詳細は [features.md](features.md) および `features/` 配下の各機能定義書を参照してください。

| # | 機能 | 概要 |
|---|------|------|
| 1 | ユーザー登録・ログイン | メール+パスワードによる登録・ログイン・ログアウト・JWT認証 |
| 2 | 読書レビュー投稿 | 書籍名・感想・星評価（1〜5）・読了／読書中ステータス・写真（最大4枚）の投稿・編集・削除 |
| 3 | いいね | レビューへのいいね追加・取り消し・いいね数表示 |
| 4 | コメント | レビューへのコメント投稿・削除・コメント数表示 |
| 5 | プロフィール・フォロー | プロフィール表示・編集・フォロー/フォロワー管理 |

---

## 3. 非機能要件・制約

| 項目 | 内容 |
|------|------|
| 感想文字数 | 制限なし（長文レビューを想定） |
| 書籍名文字数 | 最大200文字 |
| 星評価 | 1〜5の整数（必須） |
| 読書ステータス | reading（読書中）/ completed（読了）の2択（必須） |
| 写真枚数上限 | 1レビューにつき最大4枚（JPEG/PNG） |
| 画像保存先 | AWS S3（presigned URL方式でアップロード） |
| OAuth認証 | 提供しない（メール+パスワードのみ） |

---

## 4. ユーザーストーリー

詳細は [features.md](features.md) を参照してください。

- ユーザーとして、サインアップ画面からアカウントを作成してログインしたい
- ユーザーとして、読んだ本の書籍名・感想・星評価・ステータスを記録したい
- ユーザーとして、レビューに写真を添付して視覚的に記録を残したい
- ユーザーとして、全ユーザーのレビューを一覧で閲覧したい
- ユーザーとして、フォロー中のユーザーのレビューだけを絞り込んで見たい
- ユーザーとして、参考になったレビューにいいねしたい
- ユーザーとして、レビューにコメントして交流したい
- ユーザーとして、気になるユーザーをフォローして読書仲間を増やしたい
- ユーザーとして、自分のフォロー・フォロワー一覧を確認したい
